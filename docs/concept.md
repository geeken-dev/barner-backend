概要
================================================================


## コンセプト
「知の倉庫」に読み物を積み立てる。  

## やりたいこと
個人が読みたいもの、読んだものを管理し記録するシステムがほしい。  
ここでいう「読み物」とは書籍に限らない。例えば、  

- 紙で作られた物理的な書籍
- 各種サービスで提供される電子書籍
- pdf形式の読み物
  - 論文
  - 取扱説明書
  - 同人誌
- Web記事
  - Qiita
  - Zenn
  - Note
  - 各社Tech Blog
- RSS
  - e.g. Google Cloud Service Health Updates
- SNS
  - e.g. Twitterの特定アカウントのまとめ

など。このような「読み物」を一元でまとめたい。  

全体としては、

- カテゴリ分け/タグ分類
- 高度な検索
- 読み物の自動収集
  - RSSフィードから新着記事を自動でリストに追加
  - 手動で探すサイトについて、定期的にチェックを促す
- 新着読み物のサジェスト
- 「後で読む」の整理
- 栞リスト

を提供したい。  
また、それぞれの読み物に対してはそれぞれ以下の機能/データを用意する。  

- タイトル/著者/ISBN/URLなど
- 読書メモ
- 栞（どこまで読んだか）
- 物理位置
  - 他人に貸している状態も管理できるようにしたい
